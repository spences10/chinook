# chinook

SvelteKit project using the
[Chinook database](https://www.sqlitetutorial.net/sqlite-sample-database/).

Uses Bun SQLite, SvelteKit v2, Svelte v5, and TailwindCSS.

Inspired by
[SvelteKit 1.0 with SQLite Tutorial](https://www.youtube.com/watch?v=iO4VUbQ6ua4)
by [Philipp Hartenfeller](https://github.com/phartenfeller)

## Features

- Search across tracks, artists, and albums
- Routes: albums, artists, customers, genres, invoices, playlists,
  tracks
